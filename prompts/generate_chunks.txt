HOW TO WRITE THE SCRIPT — TEACHER VOICE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

The script and all action_spoken fields are read by a kn-IN TTS engine.
Write them EXACTLY as a teacher would speak in a classroom.
Natural, warm, clear. Never robotic. Never like a screen reader.

THE GOLDEN RULE:
Read the math aloud in your head first. If it sounds odd, rewrite it.
A real teacher never says "sub", "superscript", "open bracket",
"close bracket", or "backslash". Neither should the script.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
SPOKEN MATH — COMPLETE REFERENCE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

SUBSCRIPTS — say the letter and number together, nothing else:
  a_1           → "a 1"
  a_2           → "a 2"
  a_n           → "a n"
  a_{n-1}       → "a n minus 1"
  a_{k+1}       → "a k plus 1"
  x_1 + x_2    → "x 1 plus x 2"
  S_n           → "S n"
  T_n           → "T n"
  NEVER say "sub" — ever

POWERS AND SQUARES — speak as a teacher would:
  a^2           → "a squared"
  a^3           → "a cubed"
  x^n           → "x to the power n"
  r^{n-1}       → "r to the power n minus 1"
  (a+b)^2       → "a plus b, whole squared"
  (a-b)^2       → "a minus b, whole squared"
  2^10          → "2 to the power 10"
  NEVER say "superscript" or "to the power of" for squares/cubes

BRACKETS — never announce them, speak the content naturally:
  (n-1)d           → "n minus 1, times d"
  (a+b)^2          → "a plus b, whole squared"
  2(a + 4d)        → "2 times a plus 4d"
  \frac{n}{2}(a+l) → "n by 2, times a plus l"
  NEVER say "open bracket" or "close bracket" — ever

FRACTIONS — speak as division or ratio:
  \frac{a}{b}       → "a by b"
  \frac{n}{2}       → "n by 2"
  \frac{a+b}{2}     → "a plus b, by 2"
  \frac{n(a+l)}{2}  → "n times a plus l, by 2"
  \frac{a}{1-r}     → "a by 1 minus r"
  NEVER say "fraction" or "over"

ROOTS:
  \sqrt{x}           → "square root of x"
  \sqrt{4}           → "square root of 4"
  \sqrt{a^2 + b^2}   → "square root of a squared plus b squared"

EQUALS AND OPERATORS — speak naturally mid-sentence:
  =          → "is" (mid-sentence) or "equals" (stating a result)
  +          → "plus"
  -          → "minus"
  \times     → "times"
  \div       → "divided by"
  \pm        → "plus or minus"
  \neq       → "is not equal to"
  \leq       → "is less than or equal to"
  \geq       → "is greater than or equal to"
  <          → "is less than"
  >          → "is greater than"
  \therefore → "therefore"
  \because   → "since" or "because"
  \Rightarrow → "so" or "which gives"
  \approx    → "is approximately"

FULL FORMULA EXAMPLES — exactly how a teacher reads them aloud:

  $a_n = a + (n-1)d$
  → "a n equals a plus n minus 1, times d"

  $S_n = \frac{n}{2}(a + l)$
  → "S n equals n by 2, times a plus l"

  $S_n = \frac{n}{2}[2a + (n-1)d]$
  → "S n equals n by 2, times 2a plus n minus 1 times d"

  $a^2 + b^2 = c^2$
  → "a squared plus b squared equals c squared"

  $d = a_{k+1} - a_k$
  → "d equals a k plus 1 minus a k"

  $l = a + (n-1)d$
  → "l equals a plus n minus 1, times d"

  $r = \frac{a_{k+1}}{a_k}$
  → "r equals a k plus 1 by a k"

  $A = \pi r^2$
  → "A equals pi r squared"

  $(a+b)^2 = a^2 + 2ab + b^2$
  → "a plus b whole squared equals a squared plus 2ab plus b squared"
