[ಮೂಲ_ಅವಧಾರಣೆ_1]
concept_id: class9_math_ch6_concept_1
concept_name: ಮೂಲ ರೇಖಾಗಣಿತದ ಪದಗಳು ಮತ್ತು ಕೋನಗಳ ವಿಧಗಳು (Basic Geometric Terms and Types of Angles)
domain: ರೇಖಾಗಣಿತ

[ವ್ಯಾಖ್ಯೆ]
- **ರೇಖಾಖಂಡ (Line Segment):** ಎರಡು ಅಂತ್ಯಬಿಂದುಗಳನ್ನು ಹೊಂದಿರುವ ಸರಳರೇಖೆಯ ಒಂದು ಭಾಗ. ಇದನ್ನು `\overline{AB}` ಎಂದು ಸೂಚಿಸಲಾಗುತ್ತದೆ ಮತ್ತು ಅದರ ಉದ್ದವನ್ನು `AB` ಎಂದು ಬರೆಯಲಾಗುತ್ತದೆ.
- **ಕಿರಣ (Ray):** ಒಂದು ಅಂತ್ಯಬಿಂದುವನ್ನು ಹೊಂದಿದ್ದು, ಇನ್ನೊಂದು ಬದಿಯಲ್ಲಿ ಅನಂತವಾಗಿ ಚಲಿಸುವ ಸರಳರೇಖೆಯ ಒಂದು ಭಾಗ. ಇದನ್ನು `\vec{AB}` ಎಂದು ಸೂಚಿಸಲಾಗುತ್ತದೆ.
- **ಸರಳರೇಖೆ (Line):** ಯಾವುದೇ ಅಂತ್ಯಬಿಂದುಗಳಿಲ್ಲದೆ ಎರಡೂ ದಿಕ್ಕುಗಳಲ್ಲಿ ಅನಂತವಾಗಿ ವಿಸ್ತರಿಸುವ ಬಿಂದುಗಳ ಸಮೂಹ. ಇದನ್ನು `\overleftrightarrow{AB}` ಅಥವಾ `l, m, n` ಮುಂತಾದ ಸಣ್ಣ ಅಕ್ಷರಗಳಿಂದ ಸೂಚಿಸಲಾಗುತ್ತದೆ.
- **ಏಕರೇಖಾಗತ ಬಿಂದುಗಳು (Collinear Points):** ಮೂರು ಅಥವಾ ಅದಕ್ಕಿಂತ ಹೆಚ್ಚು ಬಿಂದುಗಳು ಒಂದೇ ಸರಳರೇಖೆಯ ಮೇಲೆ ಇದ್ದರೆ, ಅವುಗಳನ್ನು ಏಕರೇಖಾಗತ ಬಿಂದುಗಳು ಎನ್ನುತ್ತಾರೆ.
- **ಏಕರೇಖಾಗತವಲ್ಲದ ಬಿಂದುಗಳು (Non-collinear Points):** ಒಂದೇ ಸರಳರೇಖೆಯ ಮೇಲೆ ಇಲ್ಲದ ಬಿಂದುಗಳನ್ನು ಏಕರೇಖಾಗತವಲ್ಲದ ಬಿಂದುಗಳು ಎನ್ನುತ್ತಾರೆ.
- **ಕೋನ (Angle):** ಒಂದೇ ಆರಂಭಿಕ ಬಿಂದುವಿನಿಂದ (ಶೃಂಗ) ಹೊರಹೊಮ್ಮುವ ಎರಡು ಕಿರಣಗಳ (ಬಾಹುಗಳು) ನಡುವಿನ ಅಳತೆ.
- **ಪೂರಕ ಕೋನಗಳು (Complementary Angles):** ಎರಡು ಕೋನಗಳ ಮೊತ್ತವು 90° ಆಗಿದ್ದರೆ, ಅವುಗಳನ್ನು ಪೂರಕ ಕೋನಗಳು ಎನ್ನುತ್ತಾರೆ.
- **ಪರಿಪೂರಕ ಕೋನಗಳು (Supplementary Angles):** ಎರಡು ಕೋನಗಳ ಮೊತ್ತವು 180° ಆಗಿದ್ದರೆ, ಅವುಗಳನ್ನು ಪರಿಪೂರಕ ಕೋನಗಳು ಎನ್ನುತ್ತಾರೆ.

[ಅಂಶಪಟ್ಟಿ]
property_name: ಕೋನಗಳ ವಿಧಗಳು (Types of Angles)
properties:
  - name: ಲಘುಕೋನ (Acute Angle)
    description: 0° ಗಿಂತ ಹೆಚ್ಚು ಮತ್ತು 90° ಗಿಂತ ಕಡಿಮೆ ಇರುವ ಕೋನ. `0° < \theta < 90°`
  - name: ಲಂಬಕೋನ (Right Angle)
    description: ನಿಖರವಾಗಿ 90° ಇರುವ ಕೋನ. `\theta = 90°`
  - name: ಅಧಿಕ ಕೋನ (Obtuse Angle)
    description: 90° ಗಿಂತ ಹೆಚ್ಚು ಮತ್ತು 180° ಗಿಂತ ಕಡಿಮೆ ಇರುವ ಕೋನ. `90° < \theta < 180°`
  - name: ಸರಳಕೋನ (Straight Angle)
    description: ನಿಖರವಾಗಿ 180° ಇರುವ ಕೋನ. ಇದು ಒಂದು ಸರಳರೇಖೆಯನ್ನು ರೂಪಿಸುತ್ತದೆ. `\theta = 180°`
  - name: ಸರಳಾಧಿಕ ಕೋನ (Reflex Angle)
    description: 180° ಗಿಂತ ಹೆಚ್ಚು ಮತ್ತು 360° ಗಿಂತ ಕಡಿಮೆ ಇರುವ ಕೋನ. `180° < \theta < 360°`

[ವಿಶ್ಲೇಷಣೆ]
ಈ ಮೂಲಭೂತ ವ್ಯಾಖ್ಯೆಗಳು ರೇಖಾಗಣಿತದ ಮುಂದಿನ ಎಲ್ಲಾ ಪ್ರಮೇಯಗಳು ಮತ್ತು ಸಮಸ್ಯೆಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು ಅಡಿಪಾಯವನ್ನು ಒದಗಿಸುತ್ತವೆ. ಕೋನಗಳ ವರ್ಗೀಕರಣ ಮತ್ತು ಅವುಗಳ ಸಂಬಂಧಗಳನ್ನು (ಪೂರಕ, ಪರಿಪೂರಕ) ಸ್ಪಷ್ಟವಾಗಿ ತಿಳಿದುಕೊಳ್ಳುವುದು ಅತ್ಯಗತ್ಯ.

[ಸಂಬಂಧ]
ಈ ವ್ಯಾಖ್ಯೆಗಳು 'ಕೋನಗಳ ಜೋಡಿಗಳು' ಮತ್ತು 'ಸಮಾಂತರ ರೇಖೆಗಳು' ಎಂಬ ಮುಂದಿನ ಅವಧಾರಣೆಗಳಿಗೆ ಆಧಾರವಾಗಿವೆ. ಸರಳಕೋನದ ಪರಿಕಲ್ಪನೆಯು ಸರಳಯುಗ್ಮ ಪ್ರಮೇಯಕ್ಕೆ ನೇರವಾಗಿ ಸಂಬಂಧಿಸಿದೆ.

[ಶೈಕ್ಷಣಿಕ_ದೃಷ್ಟಿಕೋನ]
- MCQ ಸಾಧ್ಯತೆ: ಒಂದು ಕೋನವನ್ನು ನೀಡಿ ಅದರ ವಿಧವನ್ನು (ಲಘು, ಅಧಿಕ, ಇತ್ಯಾದಿ) ಗುರುತಿಸಲು ಕೇಳಬಹುದು. ಒಂದು ಕೋನದ ಪೂರಕ ಅಥವಾ ಪರಿಪೂರಕ ಕೋನವನ್ನು ಕಂಡುಹಿಡಿಯಲು ಕೇಳಬಹುದು.
- 2-ಅಂಕ ಪ್ರಶ್ನೆ: ಇಲ್ಲ.
- 5-ಅಂಕ ಪ್ರಶ್ನೆ: ಇಲ್ಲ.
- ಪ್ರಮುಖ trap: ಸರಳಾಧಿಕ ಕೋನವನ್ನು (Reflex Angle) ಸಾಮಾನ್ಯ ಕೋನದೊಂದಿಗೆ ಗೊಂದಲ ಮಾಡಿಕೊಳ್ಳುವುದು. ಉದಾಹರಣೆಗೆ, 30° ಕೋನದ ಸರಳಾಧಿಕ ಕೋನವು `360° - 30° = 330°` ಆಗಿರುತ್ತದೆ.

concept_id: class9_math_ch6_concept_2
concept_name: ಕೋನಗಳ ಜೋಡಿಗಳು (Pairs of Angles)
domain: ರೇಖಾಗಣಿತ

[ವ್ಯಾಖ್ಯೆ]
- **ಪಾರ್ಶ್ವ ಕೋನಗಳು (Adjacent Angles):** ಎರಡು ಕೋನಗಳು ಒಂದೇ ಶೃಂಗ, ಒಂದು ಸಾಮಾನ್ಯ ಬಾಹು ಹೊಂದಿದ್ದು, ಮತ್ತು ಸಾಮಾನ್ಯವಲ್ಲದ ಬಾಹುಗಳು ಸಾಮಾನ್ಯ ಬಾಹುವಿನ ಬೇರೆ ಬೇರೆ ಬದಿಗಳಲ್ಲಿದ್ದರೆ ಅವುಗಳನ್ನು ಪಾರ್ಶ್ವ ಕೋನಗಳು ಎನ್ನುತ್ತಾರೆ.
- **ಸರಳಯುಗ್ಮ (Linear Pair of Angles):** ಒಂದು ಜೋಡಿ ಪಾರ್ಶ್ವ ಕೋನಗಳ ಸಾಮಾನ್ಯವಲ್ಲದ ಬಾಹುಗಳು ಒಂದು ಸರಳರೇಖೆಯನ್ನು ಉಂಟುಮಾಡಿದರೆ, ಆ ಕೋನಗಳನ್ನು ಸರಳಯುಗ್ಮ ಎನ್ನುತ್ತಾರೆ. ಸರಳಯುಗ್ಮವನ್ನು ರಚಿಸುವ ಕೋನಗಳ ಮೊತ್ತ ಯಾವಾಗಲೂ 180° ಇರುತ್ತದೆ.
- **ಶೃಂಗಾಭಿಮುಖ ಕೋನಗಳು (Vertically Opposite Angles):** ಎರಡು ಸರಳರೇಖೆಗಳು ಪರಸ್ಪರ ಛೇದಿಸಿದಾಗ ಉಂಟಾಗುವ ಅಭಿಮುಖ ಕೋನಗಳ ಜೋಡಿ. ಈ ಕೋನಗಳು ಯಾವಾಗಲೂ ಸಮನಾಗಿರುತ್ತವೆ.

[ಪ್ರಮೇಯ]
- **ಆಧಾರ ಪ್ರತಿಜ್ಞೆ 6.1 (Axiom 6.1 - ಸರಳಯುಗ್ಮ ಆಧಾರ ಪ್ರತಿಜ್ಞೆ):**
  statement: "ಒಂದು ಸರಳರೇಖೆಯ ಮೇಲೆ ಒಂದು ಕಿರಣವು ನಿಂತಾಗ ಉಂಟಾಗುವ ಪಾರ್ಶ್ವ ಕೋನಗಳ ಮೊತ್ತವು 180° ಆಗಿರುತ್ತದೆ."
  proof_sketch: ಇದು ಸ್ವಯಂಸಿದ್ಧ ಸತ್ಯವಾದುದರಿಂದ, ಇದಕ್ಕೆ ಸಾಧನೆ ಅಗತ್ಯವಿಲ್ಲ.

- **ಆಧಾರ ಪ್ರತಿಜ್ಞೆ 6.2 (Axiom 6.2 - ಸರಳಯುಗ್ಮ ಆಧಾರ ಪ್ರತಿಜ್ಞೆಯ ವಿಲೋಮ):**
  statement: "ಎರಡು ಪಾರ್ಶ್ವ ಕೋನಗಳ ಮೊತ್ತವು 180° ಆಗಿದ್ದರೆ, ಆ ಕೋನಗಳ ಸಾಮಾನ್ಯವಲ್ಲದ ಬಾಹುಗಳು ಒಂದು ಸರಳರೇಖೆಯನ್ನು ಉಂಟುಮಾಡುತ್ತವೆ."
  converse: ಇದು ಆಧಾರ ಪ್ರತಿಜ್ಞೆ 6.1 ರ ವಿಲೋಮವಾಗಿದೆ.

- **ಪ್ರಮೇಯ 6.1 (Theorem 6.1):**
  theorem_name: Vertically Opposite Angles Theorem
  statement: "ಎರಡು ಸರಳರೇಖೆಗಳು ಪರಸ್ಪರ ಛೇದಿಸಿದಾಗ ಉಂಟಾಗುವ ಶೃಂಗಾಭಿಮುಖ ಕೋನಗಳು ಸಮವಾಗಿರುತ್ತವೆ."
  proof_sketch:
    1. AB ಮತ್ತು CD ರೇಖೆಗಳು O ನಲ್ಲಿ ಛೇದಿಸಲಿ.
    2. ಕಿರಣ OA, ರೇಖೆ CD ಮೇಲೆ ನಿಂತಿದೆ. ಆದ್ದರಿಂದ, `∠AOC + ∠AOD = 180°` (ಸರಳಯುಗ್ಮ). (i)
    3. ಕಿರಣ OD, ರೇಖೆ AB ಮೇಲೆ ನಿಂತಿದೆ. ಆದ್ದರಿಂದ, `∠AOD + ∠BOD = 180°` (ಸರಳಯುಗ್ಮ). (ii)
    4. (i) ಮತ್ತು (ii) ರಿಂದ, `∠AOC + ∠AOD = ∠AOD + ∠BOD`.
    5. ಎರಡೂ ಬದಿಗಳಿಂದ `∠AOD` ಯನ್ನು ಕಳೆದಾಗ, `∠AOC = ∠BOD` ಎಂದು ಸಿಗುತ್ತದೆ.
    6. ಇದೇ ರೀತಿ `∠AOD = ∠BOC` ಎಂದು ಸಾಧಿಸಬಹುದು.

  steps:
    - step: 1
      action: "∠POR ಮತ್ತು ∠ROQ ಗಳು ಸರಳಯುಗ್ಮವನ್ನು ಉಂಟುಮಾಡುತ್ತವೆ. ಆದ್ದರಿಂದ, ಅವುಗಳ ಮೊತ್ತ 180°."
      justification: "ಸರಳಯುಗ್ಮ ಆಧಾರ ಪ್ರತಿಜ್ಞೆ"
      latex: "∠POR + ∠ROQ = 180°"
    - step: 2
      action: "ಕೋನಗಳನ್ನು ಅನುಪಾತಕ್ಕೆ ಅನುಗುಣವಾಗಿ 5x ಮತ್ತು 7x ಎಂದು ತೆಗೆದುಕೊಳ್ಳಿ."
      justification: "ದತ್ತ ಅನುಪಾತ"
      latex: "5x + 7x = 180°"
    - step: 3
      action: "x ನ ಬೆಲೆಯನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ."
      justification: "ಬೀಜಗಣಿತದ ಸಮೀಕರಣ"
      latex: "12x = 180° \implies x = 15°"
    - step: 4
      action: "ಪ್ರತಿ ಕೋನದ ಅಳತೆಯನ್ನು ಲೆಕ್ಕ ಹಾಕಿ."
      justification: "x ನ ಬೆಲೆಯನ್ನು ಆದೇಶಿಸುವುದು"
      latex: "∠POR = 5x = 5 \times 15° = 75° \newline ∠ROQ = 7x = 7 \times 15° = 105°"
    - step: 5
      action: "ಶೃಂಗಾಭಿಮುಖ ಕೋನಗಳನ್ನು ಗುರುತಿಸಿ."
      justification: "ಪ್ರಮೇಯ 6.1"
      latex: "∠SOQ = ∠POR = 75° \newline ∠POS = ∠ROQ = 105°"
  result: "ಕೋನಗಳು 75°, 105°, 75°, ಮತ್ತು 105° ಆಗಿವೆ."

  steps:
    - step: 1
      action: "∠POS ಮತ್ತು ∠SOQ ಗಳು ಸರಳಯುಗ್ಮವಾಗಿವೆ."
      justification: "ದತ್ತಾಂಶ"
      latex: "∠POS + ∠SOQ = 180° \implies x + ∠SOQ = 180° \implies ∠SOQ = 180° - x"
    - step: 2
      action: "OR, ∠POS ನ ಕೋನಾರ್ಧಕವಾಗಿದೆ."
      justification: "ದತ್ತಾಂಶ"
      latex: "∠ROS = \frac{1}{2} ∠POS = \frac{x}{2}"
    - step: 3
      action: "OT, ∠SOQ ನ ಕೋನಾರ್ಧಕವಾಗಿದೆ."
      justification: "ದತ್ತಾಂಶ"
      latex: "∠SOT = \frac{1}{2} ∠SOQ = \frac{1}{2}(180° - x) = 90° - \frac{x}{2}"
    - step: 4
      action: "∠ROT ಯನ್ನು ಕಂಡುಹಿಡಿಯಲು ∠ROS ಮತ್ತು ∠SOT ಯನ್ನು ಕೂಡಿಸಿ."
      justification: "ಕೋನಗಳ ಸಂಕಲನ"
      latex: "∠ROT = ∠ROS + ∠SOT = \frac{x}{2} + (90° - \frac{x}{2}) = 90°"
  result: "∠ROT ಯ ಅಳತೆಯು 90° ಆಗಿದೆ."

domain: ರೇಖಾಗಣಿತ

[ವ್ಯಾಖ್ಯೆ]
- **ಸಮಾಂತರ ರೇಖೆಗಳು (Parallel Lines):** ಒಂದೇ ಸಮತಲದಲ್ಲಿರುವ ಎರಡು ರೇಖೆಗಳು ಪರಸ್ಪರ ಎಂದಿಗೂ ಛೇದಿಸದಿದ್ದರೆ, ಅವುಗಳನ್ನು ಸಮಾಂತರ ರೇಖೆಗಳು ಎನ್ನುತ್ತಾರೆ. ಅವುಗಳ ನಡುವಿನ ಲಂಬ ದೂರವು ಯಾವಾಗಲೂ ಸ್ಥಿರವಾಗಿರುತ್ತದೆ.
- **ಛೇದಕ (Transversal):** ಎರಡು ಅಥವಾ ಹೆಚ್ಚು ರೇಖೆಗಳನ್ನು ವಿಭಿನ್ನ ಬಿಂದುಗಳಲ್ಲಿ ಛೇದಿಸುವ ಒಂದು ಸರಳರೇಖೆ.

[ಪ್ರಮೇಯ]
- **ಪ್ರಮೇಯ 6.6 (Theorem 6.6):**
  theorem_name: Transitivity of Parallel Lines
  statement: "ಒಂದೇ ರೇಖೆಗೆ ಸಮಾಂತರವಾಗಿರುವ ರೇಖೆಗಳು ಪರಸ್ಪರ ಸಮಾಂತರವಾಗಿರುತ್ತವೆ."
  proof_sketch:
    1. `l || n` ಮತ್ತು `m || n` ಎಂದು ದತ್ತವಿದೆ. ನಾವು `l || m` ಎಂದು ಸಾಧಿಸಬೇಕು.
    2. ಒಂದು ಛೇದಕ `t` ಯನ್ನು ಎಳೆಯಿರಿ.
    3. `l || n` ಆಗಿರುವುದರಿಂದ, ಅನುರೂಪ ಕೋನಗಳು `∠1 = ∠3`. (i)
    4. `m || n` ಆಗಿರುವುದರಿಂದ, ಅನುರೂಪ ಕೋನಗಳು `∠2 = ∠3`. (ii)
    5. (i) ಮತ್ತು (ii) ರಿಂದ, `∠1 = ∠2`.
    6. ಆದರೆ `∠1` ಮತ್ತು `∠2` ಗಳು `l` ಮತ್ತು `m` ರೇಖೆಗಳಿಗೆ ಅನುರೂಪ ಕೋನಗಳಾಗಿವೆ.
    7. ಅನುರೂಪ ಕೋನಗಳ ಆಧಾರ ಪ್ರತಿಜ್ಞೆಯ ವಿಲೋಮದ ಪ್ರಕಾರ, `l || m`.
