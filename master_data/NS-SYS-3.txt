[ಮೂಲ_ಅವಧಾರಣೆ_1]
concept_id: class9_math_ch1_concept_1
concept_name: ಸಂಖ್ಯಾ ವ್ಯವಸ್ಥೆಗಳ ಪರಿಚಯ
domain: ಸಂಖ್ಯಾಶಾಸ್ತ್ರ

  [ವ್ಯಾಖ್ಯೆ]
  ಗಣಿತದಲ್ಲಿ ಬಳಸುವ ಸಂಖ್ಯೆಗಳನ್ನು ಅವುಗಳ ಗುಣಲಕ್ಷಣಗಳ ಆಧಾರದ ಮೇಲೆ ವಿವಿಧ ಗುಂಪುಗಳಾಗಿ ವರ್ಗೀಕರಿಸಲಾಗಿದೆ.
  1.  **ಸ್ವಾಭಾವಿಕ ಸಂಖ್ಯೆಗಳು (Natural Numbers, N):** ಎಣಿಕೆಯ ಸಂಖ್ಯೆಗಳನ್ನು ಸ್ವಾಭಾವಿಕ ಸಂಖ್ಯೆಗಳು ಎನ್ನುತ್ತಾರೆ. ಇವು 1 ರಿಂದ ಪ್ರಾರಂಭವಾಗಿ ಅನಂತದವರೆಗೆ ಮುಂದುವರೆಯುತ್ತವೆ.
      `N = {1, 2, 3, ...}`
  2.  **ಪೂರ್ಣ ಸಂಖ್ಯೆಗಳು (Whole Numbers, W):** ಸ್ವಾಭಾವಿಕ ಸಂಖ್ಯೆಗಳ ಗುಂಪಿಗೆ ಸೊನ್ನೆಯನ್ನು (0) ಸೇರಿಸಿದಾಗ ಪೂರ್ಣ ಸಂಖ್ಯೆಗಳ ಗಣ ಉಂಟಾಗುತ್ತದೆ.
      `W = {0, 1, 2, 3, ...}`
  3.  **ಪೂರ್ಣಾಂಕಗಳು (Integers, Z):** ಪೂರ್ಣ ಸಂಖ್ಯೆಗಳ ಗಣಕ್ಕೆ ಋಣ ಸಂಖ್ಯೆಗಳನ್ನು ಸೇರಿಸಿದಾಗ ಪೂರ್ಣಾಂಕಗಳ ಗಣ ಉಂಟಾಗುತ್ತದೆ. 'Z' ಎಂಬ ಸಂಕೇತವು 'Zahlen' ಎಂಬ ಜರ್ಮನ್ ಪದದಿಂದ ಬಂದಿದೆ, ಇದರರ್ಥ 'ಎಣಿಸು'.
      `Z = {..., -3, -2, -1, 0, 1, 2, 3, ...}`

  [ಅಂಶಪಟ್ಟಿ / Properties]
  *   ಪ್ರತಿಯೊಂದು ಸ್ವಾಭಾವಿಕ ಸಂಖ್ಯೆಯು ಒಂದು ಪೂರ್ಣ ಸಂಖ್ಯೆಯಾಗಿದೆ (`N ⊂ W`).
  *   ಪ್ರತಿಯೊಂದು ಪೂರ್ಣ ಸಂಖ್ಯೆಯು ಒಂದು ಪೂರ್ಣಾಂಕವಾಗಿದೆ (`W ⊂ Z`).
  *   ಆದರೆ, ಪ್ರತಿಯೊಂದು ಪೂರ್ಣಾಂಕವು ಪೂರ್ಣ ಸಂಖ್ಯೆಯಾಗಿರುವುದಿಲ್ಲ (ಉದಾ: -2 ಒಂದು ಪೂರ್ಣಾಂಕ, ಆದರೆ ಪೂರ್ಣ ಸಂಖ್ಯೆಯಲ್ಲ).

  [ವಿಶ್ಲೇಷಣೆ]
  ಈ ಸಂಖ್ಯಾ ಗಣಗಳು ಗಣಿತದ ಮೂಲಭೂತ ಪರಿಕಲ್ಪನೆಗಳಾಗಿವೆ. ಸಂಖ್ಯಾರೇಖೆಯ ಮೇಲೆ ಈ ಸಂಖ್ಯೆಗಳನ್ನು ಗುರುತಿಸುವುದರಿಂದ ಅವುಗಳ ನಡುವಿನ ಸಂಬಂಧವನ್ನು ದೃಶ್ಯೀಕರಿಸಲು ಸಾಧ್ಯವಾಗುತ್ತದೆ. ಸ್ವಾಭಾವಿಕ ಸಂಖ್ಯೆಗಳಿಂದ ಪೂರ್ಣಾಂಕಗಳವರೆಗೆ ಸಂಖ್ಯಾ ವ್ಯವಸ್ಥೆಯು ಹೇಗೆ ವಿಸ್ತರಿಸುತ್ತದೆ ಎಂಬುದನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವುದು ಮುಖ್ಯ.

  — MCQ ಸಾಧ್ಯತೆ:
    *   -5 ಯಾವ ಸಂಖ್ಯಾ ಗಣಕ್ಕೆ ಸೇರಿದೆ? (ಪೂರ್ಣಾಂಕ)
    *   ಕೆಳಗಿನವುಗಳಲ್ಲಿ ಯಾವುದು ಪೂರ್ಣ ಸಂಖ್ಯೆಯಾಗಿದ್ದು, ಸ್ವಾಭಾವಿಕ ಸಂಖ್ಯೆಯಲ್ಲ? (0)
    *   "ಪ್ರತಿಯೊಂದು ಪೂರ್ಣಾಂಕವು ಸ್ವಾಭಾವಿಕ ಸಂಖ್ಯೆಯಾಗಿದೆ" - ಈ ಹೇಳಿಕೆ ಸರಿಯೇ ಅಥವಾ ತಪ್ಪೇ? (ತಪ್ಪು)
  — 2-ಅಂಕ ಪ್ರಶ್ನೆ: ಇಲ್ಲ.
  — 5-ಅಂಕ ಪ್ರಶ್ನೆ: ಇಲ್ಲ.
  — ಪ್ರಮುಖ trap: ಸೊನ್ನೆ (0) ಒಂದು ಪೂರ್ಣ ಸಂಖ್ಯೆ ಆದರೆ ಸ್ವಾಭಾವಿಕ ಸಂಖ್ಯೆಯಲ್ಲ ಎಂಬುದನ್ನು ಮರೆಯುವುದು. ಋಣ ಸಂಖ್ಯೆಗಳು ಪೂರ್ಣಾಂಕಗಳೇ ಹೊರತು ಪೂರ್ಣ ಸಂಖ್ಯೆಗಳಲ್ಲ.

[ಮೂಲ_ಅವಧಾರಣೆ_2]
concept_id: class9_math_ch1_concept_2
concept_name: ಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಗಳು (Rational Numbers)
domain: ಸಂಖ್ಯಾಶಾಸ್ತ್ರ

  [ವ್ಯಾಖ್ಯೆ]
  ಒಂದು ಸಂಖ್ಯೆಯನ್ನು `p/q` ರೂಪದಲ್ಲಿ ವ್ಯಕ್ತಪಡಿಸಬಹುದಾದರೆ, ಅದನ್ನು ಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆ (Rational Number) ಎಂದು ಕರೆಯಲಾಗುತ್ತದೆ. ಇಲ್ಲಿ `p` ಮತ್ತು `q` ಪೂರ್ಣಾಂಕಗಳಾಗಿದ್ದು, `q` ಸೊನ್ನೆಗೆ ಸಮವಾಗಿರಬಾರದು (`q ≠ 0`).
  'Rational' ಎಂಬ ಪದವು 'Ratio' (ಅನುಪಾತ) ದಿಂದ ಬಂದಿದೆ. ಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಗಳ ಗಣವನ್ನು 'Q' (Quotient ಪದದಿಂದ) ಸಂಕೇತದಿಂದ ಸೂಚಿಸಲಾಗುತ್ತದೆ.
  ಉದಾಹರಣೆಗಳು: `1/2`, `-3/4`, `5` (ಇದನ್ನು `5/1` ಎಂದು ಬರೆಯಬಹುದು), `0` (ಇದನ್ನು `0/1` ಎಂದು ಬರೆಯಬಹುದು).

  [ಅಂಶಪಟ್ಟಿ / Properties]
  1.  **ಸಮಾನ ಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಗಳು (Equivalent Rational Numbers):** ಒಂದು ಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಯ ಅಂಶ ಮತ್ತು ಛೇದವನ್ನು ಒಂದೇ ಶೂನ್ಯವಲ್ಲದ ಪೂರ್ಣಾಂಕದಿಂದ ಗುಣಿಸಿದಾಗ ಅಥವಾ ಭಾಗಿಸಿದಾಗ, ಸಮಾನವಾದ ಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆ ದೊರೆಯುತ್ತದೆ.
      ಉದಾಹರಣೆ: `1/2 = 2/4 = 10/20`.
  2.  **ಸಾಂದ್ರತೆಯ ಗುಣ (Density Property):** ಯಾವುದೇ ಎರಡು ವಿಭಿನ್ನ ಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಗಳ ನಡುವೆ ಅಪರಿಮಿತ ಸಂಖ್ಯೆಯ ಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಗಳಿರುತ್ತವೆ.
  3.  **ಸಂಖ್ಯಾ ಗಣಗಳ ಸಂಬಂಧ:** ಎಲ್ಲಾ ಸ್ವಾಭಾವಿಕ ಸಂಖ್ಯೆಗಳು, ಪೂರ್ಣ ಸಂಖ್ಯೆಗಳು ಮತ್ತು ಪೂರ್ಣಾಂಕಗಳು ಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಗಳಾಗಿವೆ. (`N ⊂ W ⊂ Z ⊂ Q`).

  [ವಿಶ್ಲೇಷಣೆ]
  `p/q` ರೂಪದ ವ್ಯಾಖ್ಯಾನವು ಗಣಿತದಲ್ಲಿ ನಿರ್ಣಾಯಕವಾಗಿದೆ, ವಿಶೇಷವಾಗಿ `q ≠ 0` ಎಂಬ ಷರತ್ತು. ಏಕೆಂದರೆ ಸೊನ್ನೆಯಿಂದ ಭಾಗಾಕಾರವನ್ನು ವ್ಯಾಖ್ಯಾನಿಸಲಾಗಿಲ್ಲ. ಎರಡು ಸಂಖ್ಯೆಗಳ ನಡುವೆ ಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಗಳನ್ನು ಕಂಡುಹಿಡಿಯುವ ವಿಧಾನಗಳು, ಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಗಳ ಸಾಂದ್ರತೆಯನ್ನು ಪ್ರಾಯೋಗಿಕವಾಗಿ ತೋರಿಸುತ್ತವೆ.

  [ಶೈಕ್ಷಣಿಕ_ದೃಷ್ಟಿಕೋನ]
  — MCQ ಸಾಧ್ಯತೆ:
    *   ಸೊನ್ನೆ ಒಂದು ಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಯೇ? (ಹೌದು)
    *   `p/q` ರೂಪದಲ್ಲಿ `q` ಗೆ ಇರುವ ನಿರ್ಬಂಧವೇನು? (`q ≠ 0`)
  — 2-ಅಂಕ ಪ್ರಶ್ನೆ:
    *   3 ಮತ್ತು 4 ರ ನಡುವೆ ನಾಲ್ಕು ಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಗಳನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
    *   3/5 ಮತ್ತು 4/5 ರ ನಡುವೆ ಐದು ಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಗಳನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
  — 5-ಅಂಕ ಪ್ರಶ್ನೆ: ಇಲ್ಲ.
  — ಪ್ರಮುಖ trap: `q` ಸೊನ್ನೆಯಾಗಿರಬಹುದು ಎಂದು ಭಾವಿಸುವುದು.

[ಮೂಲ_ಅವಧಾರಣೆ_3]
concept_id: class9_math_ch1_concept_3
concept_name: ಅಭಾಗಲಬ್ಧ ಮತ್ತು ವಾಸ್ತವ ಸಂಖ್ಯೆಗಳು
domain: ಸಂಖ್ಯಾಶಾಸ್ತ್ರ

  [ವ್ಯಾಖ್ಯೆ]
  1.  **ಅಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಗಳು (Irrational Numbers, I):** ಒಂದು ಸಂಖ್ಯೆಯನ್ನು `p/q` ರೂಪದಲ್ಲಿ (p, q ಪೂರ್ಣಾಂಕಗಳು ಮತ್ತು q ≠ 0) ವ್ಯಕ್ತಪಡಿಸಲು ಸಾಧ್ಯವಾಗದಿದ್ದರೆ, ಅದನ್ನು ಅಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆ ಎನ್ನುತ್ತಾರೆ.
      ಉದಾಹರಣೆಗಳು: `√2`, `√3`, `√15`, `π` (ಪೈ), ಮತ್ತು `0.101101110...` ನಂತಹ ಅಂತ್ಯಗೊಳ್ಳದ ಮತ್ತು ಆವರ್ತಕವಲ್ಲದ ದಶಮಾಂಶಗಳು.
  2.  **ವಾಸ್ತವ ಸಂಖ್ಯೆಗಳು (Real Numbers, R):** ಎಲ್ಲಾ ಭಾಗಲಬ್ಧ ಮತ್ತು ಅಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಗಳ ಸಂಗ್ರಹವನ್ನು ವಾಸ್ತವ ಸಂಖ್ಯೆಗಳ ಗಣ ಎಂದು ಕರೆಯಲಾಗುತ್ತದೆ. ಸಂಖ್ಯಾರೇಖೆಯ ಮೇಲಿನ ಪ್ರತಿಯೊಂದು ಬಿಂದುವೂ ಒಂದು ಅನನ್ಯ ವಾಸ್ತವ ಸಂಖ್ಯೆಯನ್ನು ಪ್ರತಿನಿಧಿಸುತ್ತದೆ.

  [ಅಂಶಪಟ್ಟಿ / Properties]
  *   ಭಾಗಲಬ್ಧ ಮತ್ತು ಅಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಗಳ ಗಣಗಳು ಪರಸ್ಪರ ಪ್ರತ್ಯೇಕವಾಗಿವೆ (`Q ∩ I = ∅`).
  *   ಈ ಎರಡು ಗಣಗಳ ಸಂಯೋಗವೇ ವಾಸ್ತವ ಸಂಖ್ಯೆಗಳ ಗಣ (`Q ∪ I = R`).
  *   `√m` ಒಂದು ಅಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಯಾಗುತ್ತದೆ, যদি `m` ಒಂದು ಪೂರ್ಣವರ್ಗವಲ್ಲದ ಧನ ಪೂರ್ಣಾಂಕವಾಗಿದ್ದರೆ. ಉದಾಹರಣೆಗೆ, `√4 = 2` ಒಂದು ಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆ, ಆದರೆ `√5` ಒಂದು ಅಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆ.

concept_id: class9_math_ch1_concept_4
concept_name: ವಾಸ್ತವ ಸಂಖ್ಯೆಗಳ ದಶಮಾಂಶ ವಿಸ್ತರಣೆ
domain: ಸಂಖ್ಯಾಶಾಸ್ತ್ರ

  [ವ್ಯಾಖ್ಯೆ]
  ವಾಸ್ತವ ಸಂಖ್ಯೆಗಳನ್ನು ಅವುಗಳ ದಶಮಾಂಶ ವಿಸ್ತರಣೆಯ ಆಧಾರದ ಮೇಲೆ ವರ್ಗೀಕರಿಸಬಹುದು.
  1.  **ಅಂತ್ಯಗೊಳ್ಳುವ ದಶಮಾಂಶ (Terminating Decimal):** ಭಾಗಾಕಾರ ಕ್ರಿಯೆಯು ಕೆಲವು ಹಂತಗಳ ನಂತರ ಶೇಷ ಸೊನ್ನೆಯನ್ನು ನೀಡಿ ಅಂತ್ಯಗೊಂಡರೆ, ಅಂತಹ ದಶಮಾಂಶವನ್ನು ಅಂತ್ಯಗೊಳ್ಳುವ ದಶಮಾಂಶ ಎನ್ನುತ್ತಾರೆ.
      *   ಇವು ಯಾವಾಗಲೂ **ಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಗಳಾಗಿರುತ್ತವೆ**.
      *   ಉದಾಹರಣೆ: `1/8 = 0.125`.
  2.  **ಅಂತ್ಯಗೊಳ್ಳದ ಆವರ್ತ ದಶಮಾಂಶ (Non-terminating Recurring Decimal):** ಭಾಗಾಕಾರದಲ್ಲಿ ಶೇಷವು ಎಂದಿಗೂ ಸೊನ್ನೆಯಾಗದೆ, ಶೇಷದಲ್ಲಿನ ಅಂಕಿಗಳು ಒಂದು ಕ್ರಮದಲ್ಲಿ ಪುನರಾವರ್ತನೆಯಾಗುತ್ತಿದ್ದರೆ, ಭಾಗಲಬ್ಧದಲ್ಲಿನ ಅಂಕಿಗಳ ಗುಂಪು ಪುನರಾವರ್ತನೆಯಾಗುತ್ತದೆ.
      *   ಇವು ಕೂಡ **ಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಗಳಾಗಿರುತ್ತವೆ**.
      *   ಉದಾಹರಣೆ: `1/3 = 0.333... = 0.3̅`, `1/7 = 0.142857142857... = 0.142857̅`.
  3.  **ಅಂತ್ಯಗೊಳ್ಳದ ಆವರ್ತರಹಿತ ದಶಮಾಂಶ (Non-terminating Non-recurring Decimal):** ದಶಮಾಂಶ ವಿಸ್ತರಣೆಯು ಅಂತ್ಯಗೊಳ್ಳುವುದೂ ಇಲ್ಲ, ಮತ್ತು ಯಾವುದೇ ನಿರ್ದಿಷ್ಟ ಕ್ರಮದಲ್ಲಿ ಪುನರಾವರ್ತನೆಯಾಗುವುದೂ ಇಲ್ಲ.
      *   ಇವು ಯಾವಾಗಲೂ **ಅಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಗಳಾಗಿರುತ್ತವೆ**.
      *   ಉದಾಹರಣೆ: `√2 = 1.41421356...`, `π = 3.14159265...`

domain: ಬೀಜಗಣಿತ

  [ಅಂಶಪಟ್ಟಿ / Properties]
  `r` ಒಂದು ಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆ ಮತ್ತು `s` ಒಂದು ಅಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಯಾಗಿರಲಿ.
  *   ಸಂಕಲನ/ವ್ಯವಕಲನ: `r + s` ಮತ್ತು `r - s` ಗಳು ಯಾವಾಗಲೂ ಅಭಾಗಲಬ್ಧ.
  *   ಗುಣಾಕಾರ/ಭಾಗಾಕಾರ: `r * s` ಮತ್ತು `r / s` ಗಳು ಯಾವಾಗಲೂ ಅಭಾಗಲಬ್ಧ (`r ≠ 0` ಆಗಿದ್ದಾಗ).
  *   ಎರಡು ಅಭಾಗಲಬ್ಧ ಸಂಖ್ಯೆಗಳ ಮೇಲಿನ ಕ್ರಿಯೆಗಳ ಫಲಿತಾಂಶವು ಭಾಗಲಬ್ಧ ಅಥವಾ ಅಭಾಗಲಬ್ಧವಾಗಿರಬಹುದು.
      *   `√2 + (-√2) = 0` (ಭಾಗಲಬ್ಧ)
      *   `√2 * √2 = 2` (ಭಾಗಲಬ್ಧ)
      *   `√2 + √3` (ಅಭಾಗಲಬ್ಧ)

concept_id: class9_math_ch1_concept_6
concept_name: ಛೇದವನ್ನು ಅಕರಣೀಕರಿಸುವುದು (Rationalising the Denominator)
domain: ಬೀಜಗಣಿತ

  [ವ್ಯಾಖ್ಯೆ]
  ಒಂದು ಅಭಾಗಲಬ್ಧ ಛೇದವನ್ನು ಹೊಂದಿರುವ ಭಿನ್ನರಾಶಿಯನ್ನು, ಅದಕ್ಕೆ ಸಮನಾದ ಮತ್ತು ಭಾಗಲಬ್ಧ ಛೇದವನ್ನು ಹೊಂದಿರುವ ಭಿನ್ನರಾಶಿಯಾಗಿ ಪರಿವರ್ತಿಸುವ ಪ್ರಕ್ರಿಯೆಗೆ 'ಛೇದವನ್ನು ಅಕರಣೀಕರಿಸುವುದು' ಎನ್ನುತ್ತಾರೆ. ಇದು ಸಂಖ್ಯೆಯನ್ನು ಸಂಖ್ಯಾರೇಖೆಯ ಮೇಲೆ ಗುರುತಿಸಲು ಮತ್ತು ಲೆಕ್ಕಾಚಾರಗಳನ್ನು ಸುಲಭಗೊಳಿಸಲು ಸಹಾಯಕ.

  [ಅಂಶಪಟ್ಟಿ / Properties]
  **ವಿಧಾನಗಳು:**
  1.  **ರೂಪ `1/√a`:** ಅಂಶ ಮತ್ತು ಛೇದವನ್ನು `√a` ದಿಂದ ಗುಣಿಸಿ.
      `1/√a = (1 * √a) / (√a * √a) = √a / a`
  2.  **ರೂಪ `1/(a+√b)` ಅಥವಾ `1/(√a+√b)`:** ಛೇದದ ಸಂಯುಗ್ಮಿಯಿಂದ (conjugate) ಅಂಶ ಮತ್ತು ಛೇದವನ್ನು ಗುಣಿಸಿ.
      *   `a+√b` ಯ ಸಂಯುಗ್ಮಿ `a-√b`.
      *   `√a+√b` ಯ ಸಂಯುಗ್ಮಿ `√a-√b`.
      *   ಗುಣಿಸಿದಾಗ, `(x+y)(x-y) = x²-y²` ನಿತ್ಯಸಮೀಕರಣವನ್ನು ಬಳಸಲಾಗುತ್ತದೆ.
