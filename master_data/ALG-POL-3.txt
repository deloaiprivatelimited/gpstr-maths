[ಮೂಲ_ಅವಧಾರಣೆ_1]
concept_id: class10_math_ch2_concept_1
concept_name: ಬಹುಪದೋಕ್ತಿಗಳ ಪರಿಚಯ ಮತ್ತು ವಿಧಗಳು
domain: ಬೀಜಗಣಿತ

  [ವ್ಯಾಖ್ಯೆ]
  **ಬಹುಪದೋಕ್ತಿ (Polynomial):**
  ಒಂದು ಚರಾಕ್ಷರ 'x' ನಲ್ಲಿನ ಬಹುಪದೋಕ್ತಿ p(x) ಯು `p(x) = a_n x^n + a_{n-1} x^{n-1} + ... + a_1 x + a_0` ರೂಪದಲ್ಲಿರುವ ಒಂದು ಬೀಜೋಕ್ತಿಯಾಗಿದೆ. ಇಲ್ಲಿ `a_0, a_1, ..., a_n` ಗಳು ಸ್ಥಿರಾಂಕಗಳಾಗಿದ್ದು (ವಾಸ್ತವ ಸಂಖ್ಯೆಗಳು) ಮತ್ತು `a_n ≠ 0` ಆಗಿರುತ್ತದೆ. ಚರಾಕ್ಷರದ ಘಾತಾಂಕಗಳು (n, n-1, ...) ಯಾವಾಗಲೂ ಋಣಾತ್ಮಕವಲ್ಲದ ಪೂರ್ಣಾಂಕಗಳಾಗಿರಬೇಕು (non-negative integers).

  **ಬಹುಪದೋಕ್ತಿಯ ಪದಗಳು ಮತ್ತು ಸಹಗುಣಕಗಳು (Terms and Coefficients):**
  `a_n x^n`, `a_{n-1} x^{n-1}`, ಇತ್ಯಾದಿಗಳನ್ನು ಬಹುಪದೋಕ್ತಿಯ 'ಪದಗಳು' (terms) ಎಂದು ಕರೆಯಲಾಗುತ್ತದೆ. ಪ್ರತಿ ಪದದಲ್ಲಿ, ಚರಾಕ್ಷರದ ಜೊತೆಗಿನ ಸ್ಥಿರಾಂಕವನ್ನು 'ಸಹಗುಣಕ' (coefficient) ಎನ್ನುತ್ತಾರೆ. ಉದಾಹರಣೆಗೆ, `5x³ - 4x² + x - √2` ರಲ್ಲಿ, `x³` ನ ಸಹಗುಣಕ 5, `x²` ನ ಸಹಗುಣಕ -4, `x` ನ ಸಹಗುಣಕ 1.

  **ಬಹುಪದೋಕ್ತಿಯ ಮಹತ್ತಮ ಘಾತ (Degree of a Polynomial):**
  ಒಂದು ಬಹುಪದೋಕ್ತಿಯಲ್ಲಿ, ಚರಾಕ್ಷರದ ಅತ್ಯಂತ ಹೆಚ್ಚಿನ ಘಾತಸೂಚಿಯನ್ನು ಆ ಬಹುಪದೋಕ್ತಿಯ 'ಮಹತ್ತಮ ಘಾತ' ಅಥವಾ 'ಡಿಗ್ರಿ' ಎಂದು ಕರೆಯಲಾಗುತ್ತದೆ.
  *   ಉದಾಹರಣೆ: `5x³ - 4x² + x - √2` ರ ಡಿಗ್ರಿ 3 ಆಗಿದೆ.
  *   `2y² - 3y + 4` ರ ಡಿಗ್ರಿ 2 ಆಗಿದೆ.

  **ಸ್ಥಿರ ಮತ್ತು ಶೂನ್ಯ ಬಹುಪದೋಕ್ತಿಗಳು (Constant and Zero Polynomials):**
  *   **ಸ್ಥಿರ ಬಹುಪದೋಕ್ತಿ:** `p(x) = c` (ಇಲ್ಲಿ c ಒಂದು ಶೂನ್ಯವಲ್ಲದ ಸ್ಥಿರಾಂಕ) ರೂಪದ ಬಹುಪದೋಕ್ತಿಯನ್ನು ಸ್ಥಿರ ಬಹುಪದೋಕ್ತಿ ಎನ್ನುತ್ತಾರೆ. ಇದರ ಡಿಗ್ರಿ 0. ಉದಾಹರಣೆ: `p(x) = 7`.
  *   **ಶೂನ್ಯ ಬಹುಪದೋಕ್ತಿ:** ಸ್ಥಿರಾಂಕ 0 ಅನ್ನು ಶೂನ್ಯ ಬಹುಪದೋಕ್ತಿ ಎನ್ನುತ್ತಾರೆ. ಇದರ ಡಿಗ್ರಿಯನ್ನು ವ್ಯಾಖ್ಯಾನಿಸಲಾಗಿಲ್ಲ.

  [ಅಂಶಪಟ್ಟಿ / Properties]
  **ಬಹುಪದೋಕ್ತಿಗಳ ವಿಧಗಳು (ಪದಗಳ ಸಂಖ್ಯೆ ಆಧರಿಸಿ):**
  1.  **ಏಕಪದೋಕ್ತಿ (Monomial):** ಕೇವಲ ಒಂದೇ ಒಂದು ಪದವನ್ನು ಹೊಂದಿರುವ ಬಹುಪದೋಕ್ತಿ. ಉದಾಹರಣೆ: `2x`, `5x³`, `-7`.
  2.  **ದ್ವಿಪದೋಕ್ತಿ (Binomial):** ಎರಡು ಪದಗಳನ್ನು ಹೊಂದಿರುವ ಬಹುಪದೋಕ್ತಿ. ಉದಾಹರಣೆ: `x + 1`, `y³⁰ + 1`.
  3.  **ತ್ರಿಪದೋಕ್ತಿ (Trinomial):** ಮೂರು ಪದಗಳನ್ನು ಹೊಂದಿರುವ ಬಹುಪದೋಕ್ತಿ. ಉದಾಹರಣೆ: `x² + x + π`.

  **ಬಹುಪದೋಕ್ತಿಗಳ ವಿಧಗಳು (ಡಿಗ್ರಿ ಆಧರಿಸಿ):**
  1.  **ರೇಖಾತ್ಮಕ ಬಹುಪದೋಕ್ತಿ (Linear Polynomial):** ಡಿಗ್ರಿ 1 ಆಗಿರುವ ಬಹುಪದೋಕ್ತಿ. ಇದರ ಸಾಮಾನ್ಯ ರೂಪ `ax + b`, ಇಲ್ಲಿ `a, b` ಗಳು ವಾಸ್ತವ ಸಂಖ್ಯೆಗಳು ಮತ್ತು `a ≠ 0`. ಉದಾಹರಣೆ: `2x - 3`, `√3x + 5`.
  2.  **ವರ್ಗ ಬಹುಪದೋಕ್ತಿ (Quadratic Polynomial):** ಡಿಗ್ರಿ 2 ಆಗಿರುವ ಬಹುಪದೋಕ್ತಿ. ಇದರ ಸಾಮಾನ್ಯ ರೂಪ `ax² + bx + c`, ಇಲ್ಲಿ `a, b, c` ಗಳು ವಾಸ್ತವ ಸಂಖ್ಯೆಗಳು ಮತ್ತು `a ≠ 0`. ಉದಾಹರಣೆ: `2x² + 3x - 2/5`, `y² - 2`.
  3.  **ಘನ ಬಹುಪದೋಕ್ತಿ (Cubic Polynomial):** ಡಿಗ್ರಿ 3 ಆಗಿರುವ ಬಹುಪದೋಕ್ತಿ. ಇದರ ಸಾಮಾನ್ಯ ರೂಪ `ax³ + bx² + cx + d`, ಇಲ್ಲಿ `a, b, c, d` ಗಳು ವಾಸ್ತವ ಸಂಖ್ಯೆಗಳು ಮತ್ತು `a ≠ 0`. ಉದಾಹರಣೆ: `2 - x³`, `3x³ - 2x² + x - 1`.

  [ವಿಶ್ಲೇಷಣೆ]
  ಈ ಅವಧಾರಣೆಯು ಬಹುಪದೋಕ್ತಿಗಳ ಮೂಲಭೂತ ರಚನೆ ಮತ್ತು ವರ್ಗೀಕರಣವನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳಲು ಅತ್ಯಗತ್ಯ. ಬೀಜಗಣಿತದ ಮುಂದಿನ ಎಲ್ಲಾ ಪರಿಕಲ್ಪನೆಗಳಿಗೆ ಇದು ಅಡಿಪಾಯವಾಗಿದೆ. ಒಂದು ಬೀಜೋಕ್ತಿಯು ಬಹುಪದೋಕ್ತಿಯಾಗಬೇಕಾದರೆ, ಚರಾಕ್ಷರದ ಘಾತಾಂಕಗಳು ಪೂರ್ಣಾಂಕಗಳಾಗಿರಬೇಕು ಮತ್ತು ಋಣಾತ್ಮಕವಾಗಿರಬಾರದು ಎಂಬುದು ಪ್ರಮುಖ ನಿಯಮ. `x + 1/x` (ಅಂದರೆ `x + x⁻¹`) ಅಥವಾ `√x + 3` (ಅಂದರೆ `x¹/² + 3`) ಇವು ಬಹುಪದೋಕ್ತಿಗಳಲ್ಲ.

  [ಶೈಕ್ಷಣಿಕ_ದೃಷ್ಟಿಕೋನ]
  - MCQ ಸಾಧ್ಯತೆ:
    - ಕೆಳಗಿನವುಗಳಲ್ಲಿ ಯಾವುದು ಬಹುಪದೋಕ್ತಿ?
    - `p(x) = ...` ಬಹುಪದೋಕ್ತಿಯ ಡಿಗ್ರಿ ಎಷ್ಟು?
    - ಒಂದು ರೇಖಾತ್ಮಕ/ವರ್ಗ/ಘನ ಬಹುಪದೋಕ್ತಿಯನ್ನು ಗುರುತಿಸಿ.
  - 2-ಅಂಕ ಪ್ರಶ್ನೆ:
    - ಕೊಟ್ಟಿರುವ ಬಹುಪದೋಕ್ತಿಗಳನ್ನು ಅವುಗಳ ಡಿಗ್ರಿ ಅಥವಾ ಪದಗಳ ಸಂಖ್ಯೆಗೆ ಅನುಗುಣವಾಗಿ ವರ್ಗೀಕರಿಸಿ.
  - ಪ್ರಮುಖ trap: ಚರಾಕ್ಷರದ ಘಾತಾಂಕವು ಭಿನ್ನರಾಶಿ ಅಥವಾ ಋಣಾತ್ಮಕ ಸಂಖ್ಯೆಯಾಗಿರುವ ಬೀಜೋಕ್ತಿಗಳನ್ನು ಬಹುಪದೋಕ್ತಿ ಎಂದು ತಪ್ಪಾಗಿ ಗುರುತಿಸುವುದು.

[ಮೂಲ_ಅವಧಾರಣೆ_2]
concept_id: class10_math_ch2_concept_2
concept_name: ಬಹುಪದೋಕ್ತಿಯ ಶೂನ್ಯತೆಗಳು
domain: ಬೀಜಗಣಿತ

  [ವ್ಯಾಖ್ಯೆ]
  **ಬಹುಪದೋಕ್ತಿಯ ಬೆಲೆ (Value of a Polynomial):**
  `p(x)` ಒಂದು ಬಹುಪದೋಕ್ತಿಯಾಗಿದ್ದು, `k` ಒಂದು ವಾಸ್ತವ ಸಂಖ್ಯೆಯಾದರೆ, `p(x)` ನಲ್ಲಿ `x` ನ ಬದಲಿಗೆ `k` ಯನ್ನು ಆದೇಶಿಸಿದಾಗ ದೊರೆಯುವ ಬೆಲೆಯನ್ನು `x = k` ಆದಾಗ `p(x)` ನ ಬೆಲೆ ಎನ್ನುತ್ತೇವೆ ಮತ್ತು ಅದನ್ನು `p(k)` ಎಂದು ಸೂಚಿಸುತ್ತೇವೆ.

  **ಬಹುಪದೋಕ್ತಿಯ ಶೂನ್ಯತೆ (Zero of a Polynomial):**
  ಒಂದು ವಾಸ್ತವ ಸಂಖ್ಯೆ `k` ಗೆ, `p(k) = 0` ಆದರೆ, `k` ಯನ್ನು ಬಹುಪದೋಕ್ತಿ `p(x)` ನ 'ಶೂನ್ಯತೆ' (zero) ಎಂದು ಕರೆಯಲಾಗುತ್ತದೆ. ಶೂನ್ಯತೆಯನ್ನು ಬಹುಪದೋಕ್ತಿಯ 'ಮೂಲ' (root) ಎಂದೂ ಕರೆಯುತ್ತಾರೆ.

  [ಕ್ರಮಬದ್ಧ ಉದಾಹರಣೆ]
  example_id: class10_math_ch2_example_1
  problem: `p(x) = x² - 3x - 4` ಬಹುಪದೋಕ್ತಿಗೆ, `x = -1` ಮತ್ತು `x = 4` ಶೂನ್ಯತೆಗಳಾಗಿವೆಯೇ ಎಂದು ಪರೀಕ್ಷಿಸಿ.
  steps:
    - step: 1
      action: `p(x)` ನಲ್ಲಿ `x = -1` ಎಂದು ಆದೇಶಿಸಿ.
      justification: `p(-1)` ರ ಬೆಲೆಯನ್ನು ಕಂಡುಹಿಡಿಯುವುದು.
      calculation: `p(-1) = (-1)² - 3(-1) - 4 = 1 + 3 - 4 = 0`
    - step: 2
      action: `p(x)` ನಲ್ಲಿ `x = 4` ಎಂದು ಆದೇಶಿಸಿ.
      justification: `p(4)` ರ ಬೆಲೆಯನ್ನು ಕಂಡುಹಿಡಿಯುವುದು.
      calculation: `p(4) = (4)² - 3(4) - 4 = 16 - 12 - 4 = 0`
  result: `p(-1) = 0` ಮತ್ತು `p(4) = 0` ಆಗಿರುವುದರಿಂದ, -1 ಮತ್ತು 4 ಇವೆರಡೂ `x² - 3x - 4` ಬಹುಪದೋಕ್ತಿಯ ಶೂನ್ಯತೆಗಳಾಗಿವೆ.

  [ಕ್ರಮಬದ್ಧ ಉದಾಹರಣೆ]
  example_id: class10_math_ch2_example_2
  problem: `p(x) = 2x + 1` ಈ ರೇಖಾತ್ಮಕ ಬಹುಪದೋಕ್ತಿಯ ಶೂನ್ಯತೆಯನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
  steps:
    - step: 1
      action: ಶೂನ್ಯತೆಯನ್ನು ಕಂಡುಹಿಡಿಯಲು, `p(x) = 0` ಎಂದು ಸಮೀಕರಿಸಿ.
      justification: ಶೂನ್ಯತೆಯ ವ್ಯಾಖ್ಯೆಯ ಪ್ರಕಾರ, ಆ ಬೆಲೆಯಲ್ಲಿ ಬಹುಪದೋಕ್ತಿಯ ಮೌಲ್ಯ ಶೂನ್ಯವಾಗಿರುತ್ತದೆ.
      calculation: `2x + 1 = 0`
    - step: 2
      action: `x` ಗಾಗಿ ಸಮೀಕರಣವನ್ನು ಬಿಡಿಸಿ.
      justification: `x` ನ ಬೆಲೆಯೇ ಬಹುಪದೋಕ್ತಿಯ ಶೂನ್ಯತೆಯಾಗಿದೆ.
      calculation: `2x = -1` => `x = -1/2`
  result: `p(x) = 2x + 1` ರ ಶೂನ್ಯತೆಯು `x = -1/2` ಆಗಿದೆ.

  [ವಿಶ್ಲೇಷಣೆ]
  ಬಹುಪದೋಕ್ತಿಯ ಶೂನ್ಯತೆಗಳನ್ನು ಕಂಡುಹಿಡಿಯುವುದು ಎಂದರೆ, ಯಾವ `x` ಬೆಲೆಗಳಿಗೆ ಬಹುಪದೋಕ್ತಿಯ ಮೌಲ್ಯವು ಶೂನ್ಯವಾಗುತ್ತದೆ ಎಂದು ನಿರ್ಧರಿಸುವುದು. ಇದು ಬಹುಪದೋಕ್ತಿ ಸಮೀಕರಣಗಳನ್ನು (`p(x) = 0`) ಬಿಡಿಸುವುದಕ್ಕೆ ಸಮನಾಗಿದೆ.
  *   ಒಂದು ರೇಖಾತ್ಮಕ ಬಹುಪದೋಕ್ತಿ `ax + b` ಗೆ ಒಂದೇ ಒಂದು ಶೂನ್ಯತೆ ಇರುತ್ತದೆ, ಅದು `x = -b/a`.
  *   ಶೂನ್ಯವಲ್ಲದ ಸ್ಥಿರ ಬಹುಪದೋಕ್ತಿಗೆ ಯಾವುದೇ ಶೂನ್ಯತೆಗಳಿರುವುದಿಲ್ಲ.
  *   ಶೂನ್ಯ ಬಹುಪದೋಕ್ತಿಗೆ, ಪ್ರತಿಯೊಂದು ವಾಸ್ತವ ಸಂಖ್ಯೆಯೂ ಒಂದು ಶೂನ್ಯತೆಯಾಗಿರುತ್ತದೆ.

  [ಸಂಬಂಧ]
  ಈ ಅವಧಾರಣೆಯು ಅಪವರ್ತನ ಪ್ರಮೇಯಕ್ಕೆ (Factor Theorem) ನೇರವಾಗಿ ಸಂಬಂಧಿಸಿದೆ. ಒಂದು ಸಂಖ್ಯೆ `k` ಯು `p(x)` ನ ಶೂನ್ಯತೆಯಾಗಿದ್ದರೆ, `(x-k)` ಯು `p(x)` ನ ಒಂದು ಅಪವರ್ತನವಾಗಿರುತ್ತದೆ.

  [ಶೈಕ್ಷಣಿಕ_ದೃಷ್ಟಿಕೋನ]
  - MCQ ಸಾಧ್ಯತೆ:
    - `p(x) = ...` ಆದರೆ `p(k)` ಯ ಬೆಲೆ ಏನು?
    - ಕೊಟ್ಟಿರುವ ಸಂಖ್ಯೆಯು ಬಹುಪದೋಕ್ತಿಯ ಶೂನ್ಯತೆಯೇ?
  - 2-ಅಂಕ ಪ್ರಶ್ನೆ:
    - ಒಂದು ರೇಖಾತ್ಮಕ ಅಥವಾ ಸರಳ ವರ್ಗ ಬಹುಪದೋಕ್ತಿಯ ಶೂನ್ಯತೆಯನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
  - ಪ್ರಮುಖ trap: `p(0)` ಕಂಡುಹಿಡಿಯುವಾಗ, ಸ್ಥಿರಾಂಕವನ್ನು ಮಾತ್ರ ಉತ್ತರವೆಂದು ಪರಿಗಣಿಸಬೇಕು. `p(k) = 0` ಮತ್ತು `k=0` ನಡುವಿನ ವ್ಯತ್ಯಾಸವನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳದಿರುವುದು.

[ಮೂಲ_ಅವಧಾರಣೆ_3]
concept_id: class10_math_ch2_concept_3
concept_name: ಶೇಷ ಪ್ರಮೇಯ ಮತ್ತು ಅಪವರ್ತನ ಪ್ರಮೇಯ
domain: ಬೀಜಗಣಿತ

  [ಪ್ರಮೇಯ]
  theorem_name: ಶೇಷ ಪ್ರಮೇಯ (Remainder Theorem)
  statement: `p(x)` ಎಂಬುದು 1 ಅಥವಾ 1ಕ್ಕಿಂತ ಹೆಚ್ಚಿನ ಡಿಗ್ರಿ ಹೊಂದಿರುವ ಯಾವುದೇ ಬಹುಪದೋಕ್ತಿಯಾಗಿರಲಿ. `a` ಎಂಬುದು ಯಾವುದೇ ವಾಸ್ತವ ಸಂಖ್ಯೆಯಾಗಿರಲಿ. `p(x)` ನ್ನು ರೇಖಾತ್ಮಕ ಬಹುಪದೋಕ್ತಿ `(x-a)` ದಿಂದ ಭಾಗಿಸಿದಾಗ ಸಿಗುವ ಶೇಷವು `p(a)` ಆಗಿರುತ್ತದೆ.
  proof_sketch: ಭಾಗಾಕಾರದ ಅನುಪ್ರಮೇಯದ ಪ್ರಕಾರ, `p(x) = (x-a)q(x) + r(x)`. ಇಲ್ಲಿ `r(x)` ನ ಡಿಗ್ರಿಯು `(x-a)` ದ ಡಿಗ್ರಿಗಿಂತ (ಅಂದರೆ 1) ಕಡಿಮೆ ಇರಬೇಕು. ಆದ್ದರಿಂದ `r(x)` ಒಂದು ಸ್ಥಿರಾಂಕ, `r` ಆಗಿರುತ್ತದೆ. `p(x) = (x-a)q(x) + r`. ಈ ಸಮೀಕರಣದಲ್ಲಿ `x=a` ಎಂದು ಆದೇಶಿಸಿದಾಗ, `p(a) = (a-a)q(a) + r = 0 * q(a) + r = r`. ಆದ್ದರಿಂದ, ಶೇಷ `r = p(a)`.

  [ಪ್ರಮೇಯ]
  theorem_name: ಅಪವರ್ತನ ಪ್ರಮೇಯ (Factor Theorem)
  statement: `p(x)` ಎಂಬುದು 1 ಅಥವಾ 1ಕ್ಕಿಂತ ಹೆಚ್ಚಿನ ಡಿಗ್ರಿ ಹೊಂದಿರುವ ಬಹುಪದೋಕ್ತಿಯಾಗಿರಲಿ ಮತ್ತು `a` ಎಂಬುದು ಯಾವುದೇ ವಾಸ್ತವ ಸಂಖ್ಯೆಯಾಗಿರಲಿ.
  1. `p(a) = 0` ಆದರೆ, `(x-a)` ಯು `p(x)` ನ ಒಂದು ಅಪವರ್ತನವಾಗಿರುತ್ತದೆ.
  2. `(x-a)` ಯು `p(x)` ನ ಒಂದು ಅಪವರ್ತನವಾದರೆ, `p(a) = 0` ಆಗಿರುತ್ತದೆ.
  proof_sketch:
  1. ಶೇಷ ಪ್ರಮೇಯದಿಂದ, `p(x) = (x-a)q(x) + p(a)`. `p(a) = 0` ಎಂದು ಕೊಟ್ಟிருರುವುದರಿಂದ, `p(x) = (x-a)q(x)`. ಇದು `(x-a)` ಯು `p(x)` ನ ಅಪವರ್ತನ ಎಂದು ತೋರಿಸುತ್ತದೆ.
  2. `(x-a)` ಯು `p(x)` ನ ಅಪವರ್ತನವಾಗಿರುವುದರಿಂದ, `p(x) = (x-a)g(x)` (ಇಲ್ಲಿ `g(x)` ಒಂದು ಬಹುಪದೋಕ್ತಿ). `x=a` ಎಂದು ಆದೇಶಿಸಿದಾಗ, `p(a) = (a-a)g(a) = 0`.
  converse: ಈ ಪ್ರಮೇಯದ ಎರಡು ಭಾಗಗಳು ಒಂದಕ್ಕೊಂದು ವಿಲೋಮವಾಗಿವೆ.

  [ಕ್ರಮಬದ್ಧ ಉದಾಹರಣೆ]
  example_id: class10_math_ch2_example_3
  problem: `x-1` ಇದು `p(x) = 4x³ + 3x² - 4x + k` ಯ ಅಪವರ್ತನವಾಗಿದ್ದರೆ, `k` ಯ ಬೆಲೆಯನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
  steps:
    - step: 1
      action: ಅಪವರ್ತನ ಪ್ರಮೇಯವನ್ನು ಅನ್ವಯಿಸಿ. `x-1` ಅಪವರ್ತನವಾಗಿರುವುದರಿಂದ, `p(1) = 0` ಆಗಿರಬೇಕು.
      justification: ಅಪವರ್ತನ ಪ್ರಮೇಯದ ಪ್ರಕಾರ, `(x-a)` ಅಪವರ್ತನವಾದರೆ `p(a)=0`. ಇಲ್ಲಿ `a=1`.
    - step: 2
      action: `p(x)` ನಲ್ಲಿ `x=1` ಎಂದು ಆದೇಶಿಸಿ ಮತ್ತು ಅದನ್ನು 0 ಗೆ ಸಮೀಕರಿಸಿ.
      calculation: `p(1) = 4(1)³ + 3(1)² - 4(1) + k = 0`
    - step: 3
      action: `k` ಗಾಗಿ ಸಮೀಕರಣವನ್ನು ಬಿಡಿಸಿ.
      calculation: `4 + 3 - 4 + k = 0` => `3 + k = 0` => `k = -3`
  result: `k` ಯ ಬೆಲೆ -3 ಆಗಿದೆ.

  [ವಿಶ್ಲೇಷಣೆ]
  ಈ ಪ್ರಮೇಯಗಳು ಬಹುಪದೋಕ್ತಿಗಳನ್ನು ಭಾಗಿಸದೆ ಶೇಷವನ್ನು ಕಂಡುಹಿಡಿಯಲು ಮತ್ತು ಅಪವರ್ತನಗಳನ್ನು ಪರೀಕ್ಷಿಸಲು ಪ್ರಬಲ ಸಾಧನಗಳಾಗಿವೆ. ಅಪವರ್ತನ ಪ್ರಮೇಯವು ಬಹುಪದೋಕ್ತಿಗಳನ್ನು ಅಪವರ್ತಿಸಲು ಮತ್ತು ಅವುಗಳ ಶೂನ್ಯತೆಗಳನ್ನು ಕಂಡುಹಿಡಿಯಲು ವಿಶೇಷವಾಗಿ ಉಪಯುಕ್ತವಾಗಿದೆ.

  [ಶೈಕ್ಷಣಿಕ_ದೃಷ್ಟಿಕೋನ]
  - MCQ ಸಾಧ್ಯತೆ:
    - `p(x)` ನ್ನು `(x-a)` ದಿಂದ ಭಾಗಿಸಿದಾಗ ಶೇಷವೇನು?
  - 2-ಅಂಕ ಪ್ರಶ್ನೆ:
    - `(x-a)` ಯು `p(x)` ನ ಅಪವರ್ತನವೇ ಎಂದು ಪರೀಕ್ಷಿಸಿ.
    - ಅಪವರ್ತನ ಪ್ರಮೇಯ ಬಳಸಿ `k` ಯ ಬೆಲೆ ಕಂಡುಹಿಡಿಯಿರಿ.
  - ಪ್ರಮುಖ trap: `(x+a)` ದಿಂದ ಭಾಗಿಸುವಾಗ, ಶೇಷವು `p(a)` ಅಲ್ಲ, `p(-a)` ಆಗಿರುತ್ತದೆ. ಚಿಹ್ನೆಗಳಲ್ಲಿ ಗೊಂದಲ ಮಾಡಿಕೊಳ್ಳುವುದು ಸಾಮಾನ್ಯ ತಪ್ಪು.

[ಮೂಲ_ಅವಧಾರಣೆ_4]
concept_id: class10_math_ch2_concept_4
concept_name: ಬಹುಪದೋಕ್ತಿಗಳ ಅಪವರ್ತಿಸುವಿಕೆ
domain: ಬೀಜಗಣಿತ

  [ವಿಶ್ಲೇಷಣೆ]
  **ವರ್ಗ ಬಹುಪದೋಕ್ತಿಯ ಅಪವರ್ತಿಸುವಿಕೆ (ಮಧ್ಯಪದ ವಿಭಜನೆ):**
  `ax² + bx + c` ರೂಪದ ವರ್ಗ ಬಹುಪದೋಕ್ತಿಯನ್ನು ಅಪವರ್ತಿಸಲು, ಮಧ್ಯಪದ `bx` ನ್ನು `px + qx` ಎಂದು ವಿಭಜಿಸಬೇಕು. ಇಲ್ಲಿ `p + q = b` ಮತ್ತು `p * q = ac` ಆಗಿರಬೇಕು. ನಂತರ ಪದಗಳನ್ನು ಗುಂಪು ಮಾಡಿ ಸಾಮಾನ್ಯ ಅಪವರ್ತನಗಳನ್ನು ತೆಗೆಯಬೇಕು.

  **ಘನ ಬಹುಪದೋಕ್ತಿಯ ಅಪವರ್ತಿಸುವಿಕೆ:**
  ಒಂದು ಘನ ಬಹುಪದೋಕ್ತಿ `p(x)` ನ್ನು ಅಪವರ್ತಿಸಲು, ಅಪವರ್ತನ ಪ್ರಮೇಯವನ್ನು ಬಳಸಲಾಗುತ್ತದೆ.
  1.  **ಊಹೆ ಮತ್ತು ಪರೀಕ್ಷೆ (Trial and Error):** `p(x)` ನ ಸ್ಥಿರಾಂಕದ ಅಪವರ್ತನಗಳನ್ನು ಪರೀಕ್ಷಿಸಿ `p(a) = 0` ಆಗುವಂತಹ ಒಂದು ಶೂನ್ಯತೆ `a` ಯನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
  2.  `p(a) = 0` ಆದರೆ, `(x-a)` ಯು `p(x)` ನ ಒಂದು ಅಪವರ್ತನವಾಗಿರುತ್ತದೆ.
  3.  `p(x)` ನ್ನು `(x-a)` ದಿಂದ ದೀರ್ಘ ಭಾಗಾಕಾರ ಮಾಡಿ, ಭಾಗಲಬ್ಧವಾಗಿ ಒಂದು ವರ್ಗ ಬಹುಪದೋಕ್ತಿ `q(x)` ನ್ನು ಪಡೆಯಿರಿ.
  4.  `q(x)` ನ್ನು ಮಧ್ಯಪದ ವಿಭಜನೆಯ ವಿಧಾನದಿಂದ ಅಪವರ್ತಿಸಿ.

  [ಕ್ರಮಬದ್ಧ ಉದಾಹರಣೆ]
  example_id: class10_math_ch2_example_4
  problem: ಮಧ್ಯಪದವನ್ನು ವಿಭಜಿಸುವ ಮೂಲಕ `6x² + 17x + 5` ನ್ನು ಅಪವರ್ತಿಸಿ.
  steps:
    - step: 1
      action: `a` ಮತ್ತು `c` ಯ ಗುಣಲಬ್ಧವನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
      justification: `p+q=b` ಮತ್ತು `pq=ac` ಆಗುವಂತೆ `p` ಮತ್ತು `q` ಗಳನ್ನು ಕಂಡುಹಿಡಿಯಬೇಕು.
      calculation: `ac = 6 * 5 = 30`. `b = 17`.
    - step: 2
      action: ಮೊತ್ತ 17 ಮತ್ತು ಗುಣಲಬ್ಧ 30 ಆಗುವ ಎರಡು ಸಂಖ್ಯೆಗಳನ್ನು (p, q) ಹುಡುಕಿ.
      justification: 30 ರ ಅಪವರ್ತನಗಳನ್ನು ಪರೀಕ್ಷಿಸಿ (1,30), (2,15), (3,10), (5,6).
      calculation: `2 + 15 = 17` ಮತ್ತು `2 * 15 = 30`. ಆದ್ದರಿಂದ `p=2`, `q=15`.
    - step: 3
      action: ಮಧ್ಯಪದ `17x` ನ್ನು `2x + 15x` ಎಂದು ವಿಭಜಿಸಿ.
      calculation: `6x² + 2x + 15x + 5`
    - step: 4
      action: ಪದಗಳನ್ನು ಗುಂಪು ಮಾಡಿ ಸಾಮಾನ್ಯ ಅಪವರ್ತನಗಳನ್ನು ಹೊರತೆಗೆಯಿರಿ.
      calculation: `2x(3x + 1) + 5(3x + 1)`
    - step: 5
      action: ಸಾಮಾನ್ಯ ದ್ವಿಪದೋಕ್ತಿ ಅಪವರ್ತನವನ್ನು ಹೊರತೆಗೆಯಿರಿ.
      calculation: `(3x + 1)(2x + 5)`
  result: `6x² + 17x + 5` ರ ಅಪವರ್ತನಗಳು `(3x + 1)` ಮತ್ತು `(2x + 5)`.

  [ಕ್ರಮಬದ್ಧ ಉದಾಹರಣೆ]
  example_id: class10_math_ch2_example_5
  problem: `x³ - 23x² + 142x - 120` ನ್ನು ಅಪವರ್ತಿಸಿ.
  steps:
    - step: 1
      action: `p(x) = x³ - 23x² + 142x - 120` ಎಂದು ಪರಿಗಣಿಸಿ. ಸ್ಥಿರಾಂಕ -120 ರ ಅಪವರ್ತನಗಳನ್ನು ಪರೀಕ್ಷಿಸಿ. `x=1` ನ್ನು ಪ್ರಯತ್ನಿಸಿ.
      justification: ಅಪವರ್ತನ ಪ್ರಮೇಯ ಬಳಸಿ ಒಂದು ಶೂನ್ಯತೆಯನ್ನು ಕಂಡುಹಿಡಿಯುವುದು.
      calculation: `p(1) = 1³ - 23(1)² + 142(1) - 120 = 1 - 23 + 142 - 120 = 143 - 143 = 0`.
    - step: 2
      action: `p(1)=0` ಆಗಿರುವುದರಿಂದ, `(x-1)` ಒಂದು ಅಪವರ್ತನವಾಗಿದೆ.
      justification: ಅಪವರ್ತನ ಪ್ರಮೇಯ.
    - step: 3
      action: `p(x)` ನ್ನು `(x-1)` ರಿಂದ ಭಾಗಿಸಿ. (ಮಧ್ಯಪದ ಹೊಂದಾಣಿಕೆ ಅಥವಾ ದೀರ್ಘ ಭಾಗಾಕಾರ).
      calculation: `x³ - x² - 22x² + 22x + 120x - 120 = x²(x-1) - 22x(x-1) + 120(x-1) = (x-1)(x² - 22x + 120)`.
    - step: 4
      action: ವರ್ಗ ಬಹುಪದೋಕ್ತಿ `x² - 22x + 120` ನ್ನು ಅಪವರ್ತಿಸಿ. ಮೊತ್ತ -22 ಮತ್ತು ಗುಣಲಬ್ಧ 120 ಆಗುವ ಸಂಖ್ಯೆಗಳು -10 ಮತ್ತು -12.
      calculation: `x² - 10x - 12x + 120 = x(x-10) - 12(x-10) = (x-10)(x-12)`.
    - step: 5
      action: ಎಲ್ಲಾ ಅಪವರ್ತನಗಳನ್ನು ಒಟ್ಟಿಗೆ ಸೇರಿಸಿ.
      calculation: `(x-1)(x-10)(x-12)`.
  result: `x³ - 23x² + 142x - 120` ರ ಅಪವರ್ತನಗಳು `(x-1)(x-10)(x-12)`.

  [ಶೈಕ್ಷಣಿಕ_ದೃಷ್ಟಿಕೋನ]
  - 2-ಅಂಕ ಪ್ರಶ್ನೆ: ಒಂದು ವರ್ಗ ಬಹುಪದೋಕ್ತಿಯನ್ನು ಅಪವರ್ತಿಸಿ.
  - 5-ಅಂಕ ಪ್ರಶ್ನೆ: ಒಂದು ಘನ ಬಹುಪದೋಕ್ತಿಯನ್ನು ಅಪವರ್ತಿಸಿ.
  - ಪ್ರಮುಖ trap: ಘನ ಬಹುಪದೋಕ್ತಿಯನ್ನು ಅಪವರ್ತಿಸುವಾಗ, ಮೊದಲ ಶೂನ್ಯತೆಯನ್ನು ಕಂಡುಹಿಡಿಯಲು ವಿಫಲವಾದರೆ ಅಥವಾ ದೀರ್ಘ ಭಾಗಾಕಾರದಲ್ಲಿ ತಪ್ಪು ಮಾಡಿದರೆ, ಪೂರ್ತಿ ಲೆಕ್ಕ ತಪ್ಪಾಗುತ್ತದೆ. ಚಿಹ್ನೆಗಳ ಬಗ್ಗೆ ಎಚ್ಚರಿಕೆ ಅಗತ್ಯ.

[ಮೂಲ_ಅವಧಾರಣೆ_5]
concept_id: class10_math_ch2_concept_5
concept_name: ಬೈಜಿಕ ನಿತ್ಯಸಮೀಕರಣಗಳು
domain: ಬೀಜಗಣಿತ

  [ವ್ಯಾಖ್ಯೆ]
  ಒಂದು ಬೈಜಿಕ ನಿತ್ಯಸಮೀಕರಣವು ಅದರಲ್ಲಿನ ಚರಾಕ್ಷರಗಳ ಎಲ್ಲಾ ಬೆಲೆಗಳಿಗೂ ಸತ್ಯವಾಗಿರುವ ಒಂದು ಬೈಜಿಕ ಸಮೀಕರಣವಾಗಿದೆ. ಇವುಗಳನ್ನು ಬೀಜೋಕ್ತಿಗಳನ್ನು ವಿಸ್ತರಿಸಲು ಮತ್ತು ಅಪವರ್ತಿಸಲು ಬಳಸಲಾಗುತ್ತದೆ.

  [ಸೂತ್ರಗಳು]
  - formula_name: Identity I
    latex: "(x+y)^2 = x^2 + 2xy + y^2"

[ಮೂಲ_ಅವಧಾರಣೆ_6]
concept_id: class10_math_ch2_concept_6
concept_name: ಶೂನ್ಯತೆಗಳ ರೇಖಾಗಣಿತೀಯ ಅರ್ಥ
domain: ಬೀಜಗಣಿತ

  [ವಿಶ್ಲೇಷಣೆ]
  ಯಾವುದೇ ಬಹುಪದೋಕ್ತಿ `p(x)` ನ ಶೂನ್ಯತೆಗಳ ರೇಖಾಗಣಿತೀಯ ಅರ್ಥವೇನೆಂದರೆ, `y = p(x)` ಸಮೀಕರಣದ ನಕ್ಷೆಯು x-ಅಕ್ಷವನ್ನು ಛೇದಿಸುವ ಬಿಂದುಗಳ x-ನಿರ್ದೇಶಾಂಕಗಳು. ಅಂದರೆ, ನಕ್ಷೆಯು x-ಅಕ್ಷವನ್ನು ಎಷ್ಟು ಬಿಂದುಗಳಲ್ಲಿ ಛೇದಿಸುತ್ತದೆಯೋ, ಆ ಬಹುಪದೋಕ್ತಿಗೆ ಅಷ್ಟು ವಾಸ್ತವ ಶೂನ್ಯತೆಗಳಿರುತ್ತವೆ.

  1.  **ರೇಖಾತ್ಮಕ ಬಹುಪದೋಕ್ತಿ (`ax+b`):**
      *   ಇದರ ನಕ್ಷೆಯು ಒಂದು ಸರಳರೇಖೆಯಾಗಿರುತ್ತದೆ.
      *   ಇದು x-ಅಕ್ಷವನ್ನು ಕೇವಲ ಒಂದೇ ಒಂದು ಬಿಂದುವಿನಲ್ಲಿ `(-b/a, 0)` ಛೇದಿಸುತ್ತದೆ.
      *   ಆದ್ದರಿಂದ, ರೇಖಾತ್ಮಕ ಬಹುಪದೋಕ್ತಿಗೆ ಒಂದೇ ಒಂದು ಶೂನ್ಯತೆ ಇರುತ್ತದೆ.

  2.  **ವರ್ಗ ಬಹುಪದೋಕ್ತಿ (`ax²+bx+c`):**
      *   ಇದರ ನಕ್ಷೆಯು 'ಪರವಲಯ' (Parabola) ಆಕಾರದಲ್ಲಿರುತ್ತದೆ. `a>0` ಆದರೆ ಮೇಲ್ಮುಖವಾಗಿ ತೆರೆದ ಮತ್ತು `a<0` ಆದರೆ ಕೆಳಮುಖವಾಗಿ ತೆರೆದ ಪರವಲಯವಿರುತ್ತದೆ.
      *   **ಸಂದರ್ಭ 1:** ನಕ್ಷೆಯು x-ಅಕ್ಷವನ್ನು ಎರಡು ವಿಭಿನ್ನ ಬಿಂದುಗಳಲ್ಲಿ ಛೇದಿಸುತ್ತದೆ. ಈ ಸಂದರ್ಭದಲ್ಲಿ, ವರ್ಗ ಬಹುಪದೋಕ್ತಿಗೆ ಎರಡು ವಿಭಿನ್ನ ಶೂನ್ಯತೆಗಳಿರುತ್ತವೆ.
      *   **ಸಂದರ್ಭ 2:** ನಕ್ಷೆಯು x-ಅಕ್ಷವನ್ನು ಒಂದೇ ಒಂದು ಬಿಂದುವಿನಲ್ಲಿ ಸ್ಪರ್ಶಿಸುತ್ತದೆ. ಈ ಸಂದರ್ಭದಲ್ಲಿ, ವರ್ಗ ಬಹುಪದೋಕ್ತಿಗೆ ಎರಡು ಸಮನಾದ ಶೂನ್ಯತೆಗಳು (ಅಂದರೆ, ಒಂದೇ ಶೂನ್ಯತೆ) ಇರುತ್ತವೆ.
      *   **ಸಂದರ್ಭ 3:** ನಕ್ಷೆಯು x-ಅಕ್ಷವನ್ನು ಛೇದಿಸುವುದೇ ಇಲ್ಲ. ಈ ಸಂದರ್ಭದಲ್ಲಿ, ವರ್ಗ ಬಹುಪದೋಕ್ತಿಗೆ ಯಾವುದೇ ವಾಸ್ತವ ಶೂನ್ಯತೆಗಳಿರುವುದಿಲ್ಲ.

  3.  **ಘನ ಬಹುಪದೋಕ್ತಿ (`ax³+bx²+cx+d`):**
      *   ಇದರ ನಕ್ಷೆಯು x-ಅಕ್ಷವನ್ನು ಗರಿಷ್ಠ 3 ಬಿಂದುಗಳಲ್ಲಿ ಛೇದಿಸಬಹುದು.
      *   ಆದ್ದರಿಂದ, ಘನ ಬಹುಪದೋಕ್ತಿಗೆ ಗರಿಷ್ಠ 3 ಶೂನ್ಯತೆಗಳಿರಬಹುದು.

  **ಸಾಮಾನ್ಯ ನಿಯಮ:** `n` ಡಿಗ್ರಿ ಹೊಂದಿರುವ ಬಹುಪದೋಕ್ತಿಗೆ ಗರಿಷ್ಠ `n` ಶೂನ್ಯತೆಗಳಿರಬಹುದು.

  [ವಿಶ್ಲೇಷಣೆ]
  ಒಂದು ಬಹುಪದೋಕ್ತಿಯ ಶೂನ್ಯತೆಗಳು ಮತ್ತು ಅದರ ಸಹಗುಣಕಗಳ ನಡುವೆ ಒಂದು ನಿರ್ದಿಷ್ಟ ಸಂಬಂಧವಿದೆ. ಈ ಸಂಬಂಧವನ್ನು ಬಳಸಿಕೊಂಡು, ಶೂನ್ಯತೆಗಳನ್ನು ತಿಳಿಯದೆ ಅವುಗಳ ಮೊತ್ತ ಮತ್ತು ಗುಣಲಬ್ಧವನ್ನು ಕಂಡುಹಿಡಿಯಬಹುದು.

  `p(x) = ax² + bx + c` ಯ ಶೂನ್ಯತೆಗಳು `α` ಮತ್ತು `β` ಆಗಿದ್ದರೆ,

    latex: "\\alpha + \\beta = -\\frac{b}{a} = -\\frac{x \\text{ ನ ಸಹಗುಣಕ}}{x^2 \\text{ ನ ಸಹಗುಣಕ}}"
  - formula_name: Product of Zeroes (Quadratic)
    latex: "\\alpha \\beta = \\frac{c}{a} = \\frac{\\text{ಸ್ಥಿರಾಂಕ}}{x^2 \\text{ ನ ಸಹಗುಣಕ}}"

    latex: "\\alpha + \\beta + \\gamma = -\\frac{b}{a} = -\\frac{x^2 \\text{ ನ ಸಹಗುಣಕ}}{x^3 \\text{ ನ ಸಹಗುಣಕ}}"

      action: ಶೂನ್ಯತೆಗಳ ಮೊತ್ತವನ್ನು ನೇರವಾಗಿ ಮತ್ತು ಸೂತ್ರದ ಮೂಲಕ ಲೆಕ್ಕ ಹಾಕಿ.

      action: ಶೂನ್ಯತೆಗಳ ಗುಣಲಬ್ಧವನ್ನು ನೇರವಾಗಿ ಮತ್ತು ಸೂತ್ರದ ಮೂಲಕ ಲೆಕ್ಕ ಹಾಕಿ.
      justification: ಸಂಬಂಧವನ್ನು ತಾಳೆ ನೋಡುವುದು.
      calculation:
        - ನೇರ ಗುಣಲಬ್ಧ: `αβ = (-2) * (-5) = 10`.
        - ಸೂತ್ರದ ಮೂಲಕ: `c/a = 10/1 = 10`.
  result: ಶೂನ್ಯತೆಗಳ ಮೊತ್ತ ಮತ್ತು ಗುಣಲಬ್ಧ ಎರಡೂ ಸಂದರ್ಭಗಳಲ್ಲಿ ಸಮನಾಗಿರುವುದರಿಂದ ಸಂಬಂಧವನ್ನು ತಾಳೆ ನೋಡಲಾಗಿದೆ.

  [ಕ್ರಮಬದ್ಧ ಉದಾಹರಣೆ]
  example_id: class10_math_ch2_example_9
  problem: ಶೂನ್ಯತೆಗಳ ಮೊತ್ತ -3 ಮತ್ತು ಗುಣಲಬ್ಧ 2 ಆಗಿರುವ ಒಂದು ವರ್ಗ ಬಹುಪದೋಕ್ತಿಯನ್ನು ಕಂಡುಹಿಡಿಯಿರಿ.
  steps:
    - step: 1
      action: ಅಪೇಕ್ಷಿತ ಬಹುಪದೋಕ್ತಿ `ax² + bx + c` ಆಗಿರಲಿ ಮತ್ತು ಅದರ ಶೂನ್ಯತೆಗಳು `α` ಮತ್ತು `β` ಆಗಿರಲಿ.
      justification: ಸಾಮಾನ್ಯ ರೂಪವನ್ನು ಪರಿಗಣಿಸುವುದು.
    - step: 2
      action: ಕೊಟ್ಟಿರುವ ದತ್ತಾಂಶವನ್ನು ಸೂತ್ರಗಳಿಗೆ ಸಮೀಕರಿಸಿ.
      calculation: `α + β = -3 = -b/a` ಮತ್ತು `αβ = 2 = c/a`.
    - step: 3
      action: `a, b, c` ಗೆ ಸರಳವಾದ ಬೆಲೆಗಳನ್ನು ಆರಿಸಿ. `a=1` ಎಂದು ಭಾವಿಸಿದರೆ,
      calculation: `-b/1 = -3` => `b=3`. `c/1 = 2` => `c=2`.
    - step: 4
      action: `a, b, c` ಬೆಲೆಗಳನ್ನು ಸಾಮಾನ್ಯ ರೂಪದಲ್ಲಿ ಆದೇಶಿಸಿ.
      calculation: `1x² + 3x + 2` ಅಂದರೆ `x² + 3x + 2`.
  result: ಅಪೇಕ್ಷಿತ ವರ್ಗ ಬಹುಪದೋಕ್ತಿಯು `x² + 3x + 2` ಆಗಿದೆ. (ಸಾಮಾನ್ಯವಾಗಿ, `k(x² + 3x + 2)` ರೂಪದ ಯಾವುದೇ ಬಹುಪದೋಕ್ತಿ ಸರಿಹೊಂದುತ್ತದೆ).

  [ಶೈಕ್ಷಣಿಕ_ದೃಷ್ಟಿಕೋನ]
  - MCQ ಸಾಧ್ಯತೆ:
    - ಒಂದು ವರ್ಗ ಬಹುಪದೋಕ್ತಿ ಕೊಟ್ಟು ಶೂನ್ಯತೆಗಳ ಮೊತ್ತ/ಗುಣಲಬ್ಧ ಕೇಳುವುದು.
  - 2-ಅಂಕ ಪ್ರಶ್ನೆ:
    - ಶೂನ್ಯತೆಗಳನ್ನು ಕಂಡುಹಿಡಿದು ಸಂಬಂಧವನ್ನು ತಾಳೆ ನೋಡಿ.
    - ಶೂನ್ಯತೆಗಳ ಮೊತ್ತ ಮತ್ತು ಗುಣಲಬ್ಧ ಕೊಟ್ಟು ವರ್ಗ ಬಹುಪದೋಕ್ತಿಯನ್ನು ರಚಿಸಿ.
  - ಪ್ರಮುಖ trap: `-b/a` ಸೂತ್ರದಲ್ಲಿನ ಋಣಾತ್ಮಕ ಚಿಹ್ನೆಯನ್ನು ಮರೆಯುವುದು. `b` ಯು ಋಣಾತ್ಮಕವಾಗಿದ್ದಾಗ, `-b/a` ಧನಾತ್ಮಕವಾಗುತ್ತದೆ ಎಂಬುದನ್ನು ಗಮನಿಸದಿರುವುದು.
